Хочешь получить промокод в Telegram?

Бот пришлёт промо только тебе. Нажми “Да”, введи свой @ник (бот должен быть открыт) — и код улетит в личку при победе.

Убедись, что нажала Start у бота, иначе он не сможет написать.
